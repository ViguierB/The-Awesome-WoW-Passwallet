First use:
	- Edit the settings.json file -> set the right World of Warcraft path. (on windows ; folders must be seperate using '\\' not '/'
	- Run the program by clicking win_passwallet.bat for Windows or passwallet.sh for Linux. 
		- if you are running this on linux, you must install [ dialog, xdotool and bc ] and the folder '.win-dep' is not required
	- First run: Enter a password that is used for encrypt your account passwords. (only one time) for change this you must delete the file '.hidden_pass' (this action will delete all your stored password)
